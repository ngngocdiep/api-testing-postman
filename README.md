# 🧪 API Testing Project (Postman)

## 📌 Giới thiệu

Dự án kiểm thử API cho hệ thống website bán hàng.
Mục tiêu là kiểm tra tính đúng đắn của các chức năng chính và đảm bảo API hoạt động ổn định, chính xác.

---

## 🔍 Phạm vi kiểm thử

### 🔐 Auth API

* Register (Đăng ký)
* Login (Đăng nhập)

### 📦 Product API

* Lấy danh sách sản phẩm
* Kiểm tra dữ liệu trả về

---

## ⚙️ Công cụ sử dụng

* Postman (Collection, Test Script)
* GitHub (lưu trữ project)

---

## ✅ Nội dung kiểm thử

* Kiểm tra **status code** (200, 400, 401, 422...)
* Validate **response body**
* Kiểm tra **response time**

### Test case bao gồm:

* Positive cases (dữ liệu hợp lệ)
* Negative cases (dữ liệu sai)
* Boundary value (giá trị biên)
* Validate input (empty, format, length...)
* Authentication (token)
* Security test (**SQL Injection**)

---

## 🧪 Kết quả

* Thiết kế và thực hiện **50+ test cases**
* Phát hiện và báo cáo lỗi liên quan đến:

  * Validate dữ liệu
  * Logic xử lý

---

## 📂 Cấu trúc project

* `api-testing-postman.json`: Postman Collection

---

## 🚀 Hướng dẫn sử dụng

1. Mở Postman
2. Import file `api-testing-postman.json`
3. Chạy collection
4. Xem kết quả trong **Test Results**
